# Design Philosophy

The website uses a fixed left sidebar and an independent scrollable content panel to create a more immersive and structured browsing experience.

The visual direction is inspired by:

- Editorial portfolios
- Creative studio websites
- Enterprise case study presentations
- Modern analytics storytelling

The goal is to present technical and business experience in a way that feels both professional and visually engaging.
